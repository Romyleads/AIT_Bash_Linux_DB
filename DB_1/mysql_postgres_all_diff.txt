+-----------------------------+-----------------------------+-----------------------------+
|         Что отличается     |        PostgreSQL           |           MySQL            |
+-----------------------------+-----------------------------+-----------------------------+
| Создание пользователя      | CREATE USER maria WITH      | CREATE USER 'maria'@'localhost' |
|                             | PASSWORD 'secure123';       | IDENTIFIED BY 'secure123'; |
+-----------------------------+-----------------------------+-----------------------------+
| Создание базы с владельцем | CREATE DATABASE name OWNER  | CREATE DATABASE name;       |
|                             | maria;                      | (владелец не указывается)   |
+-----------------------------+-----------------------------+-----------------------------+
| Автоинкремент ID           | id SERIAL PRIMARY KEY       | id INT AUTO_INCREMENT       |
+-----------------------------+-----------------------------+-----------------------------+
| Сброс автоинкремента       | TRUNCATE ... RESTART IDENTITY | TRUNCATE ... (id сбрасывается сам) |
+-----------------------------+-----------------------------+-----------------------------+
| Подключение к базе         | В psql через \c name        | Через команду USE name;     |
+-----------------------------+-----------------------------+-----------------------------+
| Кириллица (юникод)         | UTF-8 по умолчанию          | Нужна кодировка utf8mb4     |
+-----------------------------+-----------------------------+-----------------------------+
| Владельцы объектов         | Можно указать OWNER         | Нет понятия OWNER           |
+-----------------------------+-----------------------------+-----------------------------+
